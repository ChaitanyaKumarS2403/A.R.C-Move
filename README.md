# A.R.C Move
I named the tool A.R.C Move.—it's a backronym for Automated Realtime Cursor Movement.

Easy Way!
Download the .zip from [here](https://github.com/chaitanyakumar-inf/ARC-Move/archive/refs/heads/main.zip) and extract files, save at your location, run `execute.bat`.

Get the repo:
1. Clone repo
```
git clone https://github.com/chaitanyakumar-inf/ARC-Move.git
```
3. Run `execute.bat`.
